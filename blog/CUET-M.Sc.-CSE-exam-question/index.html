CUET M.Sc. CSE Exam Questions

This page contains a list of sample questions to help you prepare for the CUET M.Sc. CSE exam. Browse through the questions and enhance your preparation!

List of "Suggested" Questions:

  1. What is the advantage of the Floyd-Warshall algorithm for solving the all-pairs shortest path problem? Why can’t Dijkstra’s or Bellman-Ford algorithms be used for this purpose?
  2. Define scan conversion. What are the issues related to unequal brightness in scan conversion?
  3. Why is a binary image used in image processing? Define Otsu thresholding. Is it a global thresholding method? Why is a histogram used in image processing?
  4. What is modulation? What is adaptive modulation?
  5. A kernel for regression is strictly monotonic and decreasing. It reaches 0 when the input feature is close to 3. Given a graph with two features and seven data points, calculate the value of y when x = 0.
  6. What is the difference between masked and unmasked interrupts?
  7. Demonstrate the insertion of a node in a binary search tree. Perform pre-order and post-order traversal of a binary search tree.
  8. Solve the given equation using a Karnaugh Map.
  9. Draw a data flow diagram for an onshore banking application.
  10. Compare normalized and denormalized data structures in the context of student data and course allotment.
  11. Compute the convolution sum of two given signal functions.
  12. Calculate the NetID, subnet, and host number for the given context.
  13. Generate three-address code for the given equation.
  14. When an operating system is isolated in memory, what are two major issues that arise?
  15. Generate a Non-Deterministic Finite Automaton (NFA) for the given language.
  16. Write code to find any repeating character in a given string.
  17. What is the difference between AI and explainable AI (XAI)? What are four significant applications of AI?
  18. What is a message digest? What is the difference between a message digest and encryption?

Note: These questions are designed to help you prepare for the CUET M.Sc. CSE entrance exam.
